这是一套完整的生产级个人博客主题INKFLOW，以深邃墨绿为主色调。以下是核心特性总结：

## 景观设计

* 字体组合：Playfair Display（衬线展示体）+ DM Sans（现代无衬线正文）+ JetBrains Mono（代码）
* 深绿主色 `#0a6640` + 荧光绿扩展 `#00c98d`，明暗双模式全面搭载
* 噪点纹理增益 + 失真光晕背景，高级亮度

## 功能优势

* 固定导航栏：滚动毛玻璃效果 + 下沉阴影
* 英雄区：突变背景浮动光效 + 特色文章反馈 + 数据统计
* 6大分类对应：悬停底部彩条动画
* 多种文章互换样式：大图精选、侧图列表、标准网格
* 侧边栏：作者简介、订阅邮件、热门文章、标签云
* 搜索蒙层：`Ctrl+K`/`Cmd+K` 快捷键
* 深度模式：一键切换，全站 CSS 变量响应
* 阅读图纸条、滚动返回顶部、数字滚动滚动、视差反馈鼠标

页面全部完成，以下是各页面的核心亮点：

* **`post-detail.html` 文章详情页** — 这是整套主题里功能最完整的页面，包含：文章英雄封面区、上游浮动操作栏（点赞/书签/评论/分享）、右侧粘性目录导航（带滚动高亮）、文章代码块一键复制、标注提示框（透明样式）、内部填充表格、点赞/收藏反应栏、分享栏、作者间、上下篇导航、文章相关、评论区（含作者回复标记）。
* **`album-list.html` 相册页** — 大图英雄相册关联+分类筛选+照片瀑布流布局+灯箱灯箱全屏预览（含左右导航、下载、分享）。
* **`link-list.html` 链接页** — 精选推荐卡（含底部进度条动画）+按分类分组展示+申请友链表单（可展开收起）+本站信息一键复制+侧边栏链接统计。
