## 📦 关于 INKFLOW 仓库

**INKFLOW** 是一套专为极客、设计师和文字工作者打造的**生产级个人博客主题模板**。

### 🛠 技术架构
本主题基于最流行的前端框架 **Bootstrap 5 (BS5)** 构建，并坚持使用**原生 JavaScript (Vanilla JS)** 进行交互开发。这种组合确保了：
* **开发极其高效**：利用 BS5 成熟的栅格系统和组件库，快速实现响应式布局。
* **性能轻量无依赖**：无需 jQuery 或复杂的现代框架运行环境，浏览器原生解析，加载速度极快。
* **高度可定制**：通过 Bootstrap 的 SCSS 变量或自定义 CSS 属性，可以轻松调整主题色调和风格。

### 🎨 设计理念 - “深邃与灵动”
INKFLOW 不仅仅是一个模板，更代表一种审美态度：
* **墨绿美学**：以深邃墨绿（`#0a6640`）为主色调，营造沉静、专注的阅读氛围。
* **荧光点缀**：搭配高饱和度的荧光绿（`#00c98d`）作为交互和强调色，如同暗夜中的螢火，灵动且充满科技感。
* **光影纹理**：全站搭载**噪点纹理增益**与**失真光晕背景**，拒绝平庸的扁平化，追求高级的亮度与质感。
* **全面双模**：原生支持**明亮/黑暗（Light/Dark）**双模式一键切换，CSS 变量全站响应，保护视力，风格随心。

### 🚀 核心特性与完整页面
本仓库包含 **10 个** 精心设计的 HTML 模板文件（涵盖首页、文章列表、详情、相册、链接、归档、分类、标签以及完整的用户登录与个人信息编辑页）。

主题集成了毛玻璃导航栏、视差滚动、Ctrl+K 快捷键搜索、粘性目录、代码一键复制、图像灯箱预览等大量高级交互功能，致力于提供最顶级的 Web 阅读体验。


这是一套完整的生产级个人博客主题INKFLOW，以深邃墨绿为主色调。以下是核心特性总结：

#### 景观设计

* 字体组合：Playfair Display（衬线展示体）+ DM Sans（现代无衬线正文）+ JetBrains Mono（代码）
* 深绿主色 `#0a6640` + 荧光绿扩展 `#00c98d`，明暗双模式全面搭载
* 噪点纹理增益 + 失真光晕背景，高级亮度

#### 功能优势

* 固定导航栏：滚动毛玻璃效果 + 下沉阴影
* 英雄区：突变背景浮动光效 + 特色文章反馈 + 数据统计
* 6大分类对应：悬停底部彩条动画
* 多种文章互换样式：大图精选、侧图列表、标准网格
* 侧边栏：作者简介、订阅邮件、热门文章、标签云
* 搜索蒙层：`Ctrl+K`/`Cmd+K` 快捷键
* 深度模式：一键切换，全站 CSS 变量响应
* 阅读图纸条、滚动返回顶部、数字滚动滚动、视差反馈鼠标
